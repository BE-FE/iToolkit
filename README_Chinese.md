iToolkit
========

iToolkit是一系列常用控件的集合，基于Riot进行构建，iToolkit拥有统一的配置方式：
例如：

    <itk-tree></itk-tree>
    itk.mount('itk-tree', config);


### 为什么要开发iToolkit？
1. 统一的调用方式,数据直接生成视图。
2. 丰富的组件库，包含了很多企业级开发常用的组件
3. 到处可用，无论原网站是采用何种技术方案，都可以加入使用
4. 无依赖，体积小，压缩后仅50K。
